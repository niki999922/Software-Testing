# Getting Start
Runs the app in the development mode.\
[Open http://localhost:3000](http://localhost:3000)

```shell
yarn start
```

Run all tests:
Available tests:
* `react-scripts test`
* `jest test`
* `jest --coverage`
* `mocha "test/**/*-mocha.test.js"`
* 4
* 5


```shell
yarn test
```

Build Application for deploying

```shell
yarn build
```
